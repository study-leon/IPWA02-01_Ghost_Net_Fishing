# IPWA02-01_Ghost_Net_Fishing
Prototyp einer Webanwendung zur Meldung und Bergung von Geisternetzen (herrenlose Fischernetze im Meer).
Das Projekt basiert auf der Aufgabenstellung „Ghost Net Fishing“ und setzt die fachlichen Anforderungen prototypisch um.

## Funktionen

- Geisternetze (auch anonym) melden

- Meldende und bergende Personen verwalten

- Anzeige aller noch zu bergenden Geisternetze

- Bergung einem Geisternetz zuordnen

- Netze als geborgen oder verschollen markieren

- Weltkartenansicht offener Geisternetze

## Technologien

- Spring Boot

- Thymeleaf

- Spring Data JPA / Hibernate

- MySQL
